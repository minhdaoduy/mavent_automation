For Setup
========================================================================================
Step#1: set Java Home
Step#2: install IDE (recommended Intellij.)
Step#3: clone code from https://github.com/minhdaoduy/mavent_automation.git
Step#4: checkout brach master (default)
Step#5: right click the test case, in this case is [Tests.TellAboutYourSelf]
==> Select [Create 'TellAboutYourSelf'...]
Step#6: click run icon in top menu of Intellij IDE for execution the automation testing
========================================================================================
